## ğŸ“Š Voice Biomarkers Analyzed

| Biomarker | Description | AD Indicator |
|-----------|-------------|--------------|
| **Jitter** | Pitch perturbation (F0 variation) | Increased = vocal instability |
| **Shimmer** | Amplitude variation | Increased = reduced neuromuscular control |
| **HNR** | Harmonic-to-Noise Ratio | Decreased = breathier voice |
| **F0 Variability** | Pitch dynamics | Decreased = monotone speech |
| **Pause Patterns** | Silence frequency/duration | Increased = word-finding difficulty |
| **Formants** | F1, F2, F3 frequencies | Reduced variability = imprecise articulation |
| **MFCC** | Mel-Frequency Cepstral Coefficients | Vocal tract shape changes |
| **Speech Rate** | Voiced/unvoiced ratio | Decreased = cognitive slowing |
